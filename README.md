# Telegram backup
A tool for telegram backup


```
Usage of telegram-backup:
  -api_hash string
    	API Hash from Telegram (default "")
  -api_id int
    	API ID from Telegram (default 0)
  -contact_name string
    	Name of contact to take backup
  -phone_number string
    	Your account phone number (default "")
  -server_ip string
    	Telegram server IP (default "")
  -server_port int
    	Telegram port IP (default 443)
```
